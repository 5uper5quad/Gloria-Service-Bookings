## Development

### Installing Dependencies

From within the root directory:

```
npm install
```

#### Running the Service

From within the root directory:
```
Server: npm start
```
```
Seeder: npm run seed
```
```
Webpack: npm run react-dev
```

load up 127.0.0.1:3000/api/listings/1/reservations in Postman to see example data being returned at /:id
